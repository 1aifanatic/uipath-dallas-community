# Dallas UiPath Community Day - HTML Presentation

This repository contains a click-through HTML presentation for:

**Stop Guessing: Choose the Right UiPath Build Path - Dallas UiPath Community Day**

Open `index.html` in a browser to present. Use the on-screen controls, click the slide, or use these keys:

- `Right`, `Space`, or `PageDown`: next slide
- `Left` or `PageUp`: previous slide
- `Home`: first slide
- `End`: final slide
- `F`: fullscreen

## Source

The event details, speaker names, speaker photos, event timing, venue, and session description are based on the official UiPath Community event page:

https://community.uipath.com/events/details/uipath-dallas-texas-presents-stop-guessing-choose-the-right-uipath-build-path-dallas-uipath-community-day-in-person-event/

The published event agenda block is empty, so the deck uses a practical two-hour run-of-show derived from the official title, description, and event window.
